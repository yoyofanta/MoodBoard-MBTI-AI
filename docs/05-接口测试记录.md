
# MoodBoard × MBTI 接口测试记录

## 一、测试说明

本次接口测试主要针对系统核心接口进行验证，包括登录注册、用户资料、情绪日记、AI 聊天、定位接口等。测试过程中通过浏览器 F12 Network 查看请求地址、请求方式、请求参数、响应状态码和返回数据，验证前后端接口调用是否正常。

## 二、测试环境

| 项目     | 内容                    |
| -------- | ----------------------- |
| 前端地址 | http://localhost:5173   |
| 后端地址 | http://localhost:8888   |
| 浏览器   | Chrome                  |
| 测试方式 | Chrome DevTools Network |
| 数据库   | H2 / MySQL              |
| 测试账号 | 游客账号 / 注册账号     |

## 三、接口测试记录

| 编号    | 模块       | 接口名称            | 请求方式   | 接口地址                                   | 测试步骤                     | 预期结果                                    | 实际结果       | 测试结论 |
| ------- | ---------- | ------------------- | ---------- | ------------------------------------------ | ---------------------------- | ------------------------------------------- | -------------- | -------- |
| API-001 | 登录注册   | 游客登录 / 注册登录 | POST       | /api/auth/register 或 /api/auth/login      | 点击游客登录                 | 返回 token，进入系统首页                    | 成功进入系统   | 通过     |
| API-002 | 用户资料   | 保存用户资料        | POST / PUT | /api/profile                               | 修改昵称、身份、年龄段后保存 | 返回成功，页面资料更新                      | 成功保存       | 通过     |
| API-003 | 情绪日记   | 保存日记            | POST       | /api/diaries                               | 选择 emoji，输入文本并保存   | 返回保存成功，数据库生成记录                | 成功保存       | 通过     |
| API-004 | 情绪日记   | 查询日记            | GET        | /api/diaries/date 或相关查询接口           | 保存后刷新页面查看回显       | 返回对应日期日记数据                        | 成功回显       | 通过     |
| API-005 | 定位       | 逆地理编码          | GET        | /api/location/reverse                      | 点击获取位置                 | 返回 province、city、district、locationName | 已返回市区信息 | 通过     |
| API-006 | AI树洞     | 日常聊天            | POST       | /api/ai/chat                               | 日常树洞发送一句话           | 返回 AI 回复内容                            | AI 正常回复    | 通过     |
| API-007 | 人格对话   | 单人格聊天          | POST       | /api/ai/chat                               | 选择 MBTI 人格并发送消息     | 返回对应人格回复                            | AI 正常回复    | 通过     |
| API-008 | 心灵盲盒   | 抽取盲盒            | GET / POST | /api/blind-box/draw                        | 点击抽一个心灵盲盒           | 返回随机人格和话题                          | 正常返回       | 通过     |
| API-009 | 心灵盲盒   | 盲盒回复            | POST       | /api/blind-box/reply                       | 在盲盒对话中回复             | 返回下一轮 AI 回复                          | 正常返回       | 通过     |
| API-010 | 多人格对战 | 创建对战会话        | POST       | /api/persona-battle-sessions               | 选择 2-4 个人格开始对话      | 返回会话 ID 和人格列表                      | 正常创建       | 通过     |
| API-011 | 多人格对战 | 发送对战消息        | POST       | /api/persona-battle-sessions/{id}/messages | 输入一句话发送               | 多个人格分别回复                            | 正常回复       | 通过     |
| API-012 | 多人格对战 | 结束并生成总结      | POST       | /api/persona-battle-sessions/{id}/finish   | 点击结束会话                 | 返回总结内容                                | 正常生成总结   | 通过     |
