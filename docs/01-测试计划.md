# MoodBoard × MBTI 项目测试计划

## 一、项目名称

MoodBoard × MBTI 轻量级情绪日记与 AI 树洞应用

## 二、测试目的

验证系统主要功能是否满足需求，包括用户登录注册、情绪日记记录、AI 树洞聊天、MBTI 人格互动、心灵盲盒、多人格对战、情绪漂流瓶、定位与图片上传等功能。

通过功能测试、接口测试、异常测试和兼容性测试，发现并记录系统缺陷，提升系统稳定性和用户体验。

## 三、测试范围

### 1. 登录注册模块
- 用户登录
- 用户注册
- 游客登录
- 首次引导信息填写
- 个人信息保存

### 2. 情绪日记模块
- 日期选择
- 多情绪 emoji 选择
- 日记文本输入
- 关键词输入
- 图片上传 / 拍照
- 位置获取
- 日记保存
- 周 / 月 / 年情绪视图展示

### 3. AI 树洞模块
- 日常树洞聊天
- 单人格聊天
- MBTI 人格切换
- 心灵盲盒
- 人格面具
- 多人格对战
- 情绪漂流瓶
- 聊天记录保存
- 清空会话

### 4. 接口测试
- 登录注册接口
- 用户资料接口
- 日记接口
- 聊天接口
- 漂流瓶接口
- 心灵盲盒接口
- 多人格对战接口
- 定位接口

### 5. 异常测试
- 空输入
- 超长文本
- 未登录访问
- 后端关闭
- DeepSeek API 异常
- 高德地图 API 异常
- 图片过大
- 网络失败
